🔎 Signature Scanner

Periodic scan of known binary paths, config directories, and package manager installs.

🌐 Network Monitor

Real-time DNS & TLS SNI inspection for known LLM API domains.

  • api.anthropic.com, api.openai.com, etc.
  • Local inference server port detection
  • Catches browser-based AI usage
REAL-TIME

🧠 Behavioral Monitor

Process-level heuristics: broad file reads + outbound LLM calls + child shell execution.

  • Catches novel/unknown tools
  • Custom scripts, renamed binaries
  • The differentiator no one else detects
HEURISTIC

⚖️ Policy Engine

Evaluates detections against the who / what / how policy matrix bound to IDP groups (Okta, Entra ID).
