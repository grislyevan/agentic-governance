Engine
⚖️ Confidence Engine
  • compute_confidence() — weighted layer sum
  • Per-tool weight profiles
  • Penalty deductions (stale signals)
  • Evasion boost (renamed binaries)
  • Output: 0.0–1.0 score → High/Med/Low band

Detection Algorithm — Per-Scan Cycle
🔄 Scan Cycle Flow
┌─────────────────────────────────────────────────────┐
│ 1. SCAN for each tool in 11 scanners: │
│ scanner.scan() → ScanResult │
│ { signals[], penalties[], evidence{} } │
│ │ │
│ ▼ │
│ 2. SCORE compute_confidence(scan_result, tool) │
│ Σ (layer_weight × signal_score) − penalties │
│ + evasion_boost → 0.0 – 1.0 │
│ classify: High ≥0.7 | Med ≥0.4 | Low <0.4 │
│ │ │
│ ▼ │
│ 3. POLICY evaluate_policy(tool, confidence, class) │
│ Match rules (ENFORCE-*, NET-*, ISO-*) │
│ → decision: detect|warn|approval_required|block │
│ → matched_rules[], severity (S0–S4) │
│ │ │
│ ▼ │
│ 4. ENFORCE if decision ∈ {approval_required, block} │
│ enforcer.dispatch(tactic) │
│ → process_kill | network_block | proxy_inject │
│ │ │
│ ▼ │
│ 5. EMIT build three events: │
│ ① detection.observed │
│ ② policy.evaluated │
│ ③ enforcement.applied (if enforced) │
│ sign → validate → emit (NDJSON | HTTP) │
│ │ │
│ ▼ │
│ 6. DIFF StateDiffer compares to last scan │
│ Only emit if state changed (new/removed/changed) │
└─────────────────────────────────────────────────────┘
