   The Detec endpoint agent collects telemetry data locally on
   each endpoint to score agentic AI tool detections. Raw telemetry
   (process details, file paths, network connections) is used only
   for local confidence scoring and is NOT transmitted to the server.

   The agent transmits structured event metadata (tool name,
   confidence score, policy decisions, actor ID, endpoint ID,
   timestamps) to the Detec server.

   party unless explicitly configured by your administrator.
