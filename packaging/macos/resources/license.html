Data Collection

The Detec Agent collects telemetry data locally on this endpoint to score agentic AI tool detections. Raw telemetry (process details, file paths, network connections) is used only for local confidence scoring and is not transmitted to the server.

The Agent transmits structured event metadata (tool name, confidence score, policy decisions, actor ID, endpoint ID, timestamps) to the Detec server configured by your organization.
