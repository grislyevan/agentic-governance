Detec Agent

Version 0.3

Welcome to the Detec endpoint agent installer. This agent monitors your workstation for agentic AI tools and reports findings to your organization's Detec dashboard.

The installer will:

  • Install Detec Agent to /Applications
  • Configure the agent to start automatically at login
  • Create a menu bar icon for status monitoring

After installation, you may be prompted to grant Full Disk Access to the agent in System Settings > Privacy & Security. This permission is required for complete AI tool detection.
