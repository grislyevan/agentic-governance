Configuration

If you downloaded this package from the Detec dashboard, run the included install.sh script instead of double-clicking the .pkg directly. The script places your server configuration automatically so the agent connects with no manual setup.

For manual installs, configure the agent by placing config files in ~/Library/Application Support/Detec/:

  • agent.env (KEY=VALUE format with AGENTIC_GOV_ prefix)
  • collector.json (JSON format)

Alternatively, run detec-agent setup --api-url ... --api-key ..., set environment variables (AGENTIC_GOV_API_URL, AGENTIC_GOV_API_KEY), or store the API key in the macOS Keychain (service: detec-agent, account: api-key).

Permissions

For complete detection coverage, grant Full Disk Access to Detec Agent in System Settings. Your IT administrator may push this permission automatically via an MDM configuration profile.

Uninstalling

To remove the agent, run the uninstaller script as root (see tamper controls):

sudo bash /path/to/packaging/macos/uninstall.sh

Or manually:

  1. Quit Detec Agent from the menu bar
  2. Run: launchctl bootout gui/$(id -u)/com.detec.agent
  3. Delete /Applications/Detec Agent.app
  4. Delete ~/Library/LaunchAgents/com.detec.agent.plist
  5. (Optional) Delete ~/.agentic-gov/ and ~/Library/Application Support/Detec/
