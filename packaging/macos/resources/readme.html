Configuration

After installation, configure the agent with your organization's API endpoint. Create or edit ~/.agentic-gov/config.json:

{
  "api_url": "https://your-detec-server.example.com",
  "api_key": "your-api-key",
  "interval": 300,
  "sensitivity": "Tier0"
}

Alternatively, set environment variables AGENTIC_GOV_API_URL and AGENTIC_GOV_API_KEY, or store the API key in the macOS Keychain (service: detec-agent, account: api-key).

Permissions

For complete detection coverage, grant Full Disk Access to Detec Agent in System Settings. Your IT administrator may push this permission automatically via an MDM configuration profile.

Uninstalling

To remove the agent:

  1. Quit Detec Agent from the menu bar
  2. Run: launchctl bootout gui/$(id -u)/com.detec.agent
  3. Delete /Applications/Detec Agent.app
  4. Delete ~/Library/LaunchAgents/com.detec.agent.plist
  5. (Optional) Delete ~/.agentic-gov/ to remove state and config
