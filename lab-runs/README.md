# Lab runs

This directory holds **lab run protocols** (step-by-step capture procedures) and **results** (filled-in analysis) for empirical validation of agentic AI tool detection. Templates for tools not yet lab-run live here as well; executable capture scripts are in [scripts/](scripts/).

**Full index:** [docs/lab-runs-and-results.md](../docs/lab-runs-and-results.md) — lists every protocol and RESULTS file, capture scripts, and how they link to the playbook, INIT-43, and calibration fixtures.

**Naming:** `LAB-RUN-XXX-<description>.md` = protocol; `LAB-RUN-XXX-RESULTS.md` = results. Evidence from a run is written to `$LAB_DIR` (e.g. `~/cowork-lab/LAB-RUN-014/`), not committed.
