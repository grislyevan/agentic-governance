# Detec Product Status

Maturity of capabilities. This table is the source of truth for what is available vs experimental vs roadmap.

| Capability | Status |
|------------|--------|
| Core behavioral detections (DETEC-BEH-CORE-01, 02, 03) | Available |
| Confidence scoring and calibration | Available |
| Deterministic policy engine | Available |
| Endpoint enforcement | Available |
| Behavioral demo artifacts | Available |
| CrowdStrike enrichment | Experimental |
| Native ESF / ETW / eBPF telemetry | Experimental / roadmap |
| Dashboard and management workflows | In progress |

- **Available:** Shipped and supported for production use.
- **Experimental:** Implemented but not fully validated or supported; use with caution.
- **Roadmap:** Planned; not yet implemented or only partially implemented.
- **In progress:** Actively being completed.

See [PROGRESS.md](../PROGRESS.md) for detailed roadmap and [README](../README.md) for quick verification steps.
