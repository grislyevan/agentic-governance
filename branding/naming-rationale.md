# Detec — Naming Rationale

## The Name

**Detec** — truncated from "detect."

A modern truncation that signals the product's core function — detection — without sounding like a product name committee assembled it. The missing "t" is intentional: it signals a brand, not a dictionary entry. Follows the same naming pattern as Snyk (from "sneak"), Brex, Flickr, and Tumblr.

## Why This Name

| Product Concept | How "Detec" Maps |
|---|---|
| Multi-signal detection | Detection is the foundation — everything starts with finding the tool |
| Confidence-scored decisions | "Detec" anchors to the first step; scoring, governance, and enforcement build on top |
| Immediate clarity | Anyone hearing "Detec" knows the domain instantly — zero explanation needed |
| Modern truncation | The missing letter signals intentionality — this is a brand, not a generic label |
| SOC operator audience | Operators live in detection. The name speaks their language. |
| Extensible | "Detec Collector," "Detec Dashboard," "Detec API" — clean component naming |

## Pronunciation

**dee-TEK** — two syllables, stress on the second.

## Competitive Clearance

Searched March 2026. No tech company found using "Detec" as an exact company or product name. Neighboring names exist in the detection space (Detekt, Detecto, Detect Technologies, Dectar) but none use the exact truncation "Detec." The name is sufficiently distinct to own.

## Rejected Candidates

Over 50 names were evaluated across multiple naming approaches:

- **Mythological** (Greek, Norse, Celtic): Metis, Axiom, Theia, Tyr, Mimir, Huginn, Forseti, Saga, Themis, Argus, Tessera, Vigil, Brehon, Ogham, Dagda, Danu, Geas, Orlog, Vardr — all had existing tech company conflicts or didn't resonate.
- **Latin verbs**: Sentio, Cerno, Probo, Lustro, Specto, Aperio, Certus, Luceo, Tego, Custos, Munio, Indago, Penso — all taken by existing tech companies.
- **Descriptive compounds**: AgentWatch, AgentScope, SignalStack — rejected as "internet cops."
- **Smooth coined words**: Lirano, Nilero, Rivelo, Velora, Sovra — available but didn't connect.
- **Protec** — Protec Security (UK) exists; more crowded namespace than Detec.
